Propuesta económica

duki fases, entregables y costos

Nota

Los montos son una guía para definir alcance. El costo final depende de: número de ubicaciones activas, número de usuarios staff, reglas de precios/promociones, y nivel de hardening requerido.

Si prefieres, puedo convertir esto a una tabla cerrada en cuanto me confirmes alcance exacto (MVP) y fecha objetivo.

Fase 0 — Demo (1-2 días)

  • Objetivo: validar flujo NFC → login → vista de socio y UX “menos clicks”.
  • Entregables: demos navegables + brief/propuesta.
  • Costo: $0 (demo actual) / o fijo simbólico si se formaliza el ejercicio.

Fase 1 — MVP operativo (2-4 semanas)

  • Socios/categorías, catálogo, menús por categoría.
  • Precios dinámicos (descuento por categoría) + base para promociones.
  • Wallet: recargas + consumos + historial.
  • Cobros: efectivo/tarjeta/transferencia/wallet.
  • Multi-ubicación (2 sucursales + entrenamiento).
  • Usuarios staff + roles + auditoría básica.

Rango estimado: $35,000 – $85,000 MXN

Fase 2 — Seguridad y robustez (1-2 semanas)

  • Token temporal / one-time para mitigar enlaces compartidos.
  • Rate-limit de intentos de PIN.
  • Backups automatizados y bitácora avanzada.
  • Mejoras UX para operación (barra/caja) y performance.

Rango estimado: $15,000 – $45,000 MXN

Fase 3 — Reportes y promociones avanzadas (2-4 semanas)

  • Reportes por ubicación/categoría/producto (ventas, recargas, consumo).
  • Promos configurables (happy hour, combos, reglas por horario).
  • Historial detallado del socio (preferencias, frecuencia).

Rango estimado: $25,000 – $70,000 MXN

Opcionales

  • Diseño UI más avanzado / branding completo: $8,000 – $25,000 MXN
  • Soporte y mantenimiento mensual: $3,000 – $12,000 MXN
  • Implementación en sitio (capacitación staff): $2,000 – $10,000 MXN

Supuestos

  • El demo no es un sistema productivo: no incluye persistencia real ni validación server-side.
  • Para operar wallet/cobros se requiere backend y almacenamiento confiable.
  • Las integraciones de pago (terminal/banco) no están incluidas; se modela registro de pago.
